第二章：准备基本类库

1. 需要用到的第三方类库：

  • jQuery (http://jquery.com/)
  • SeaJs (http://seajs.org/docs/)

把 jQuery 和 SeaJs 放到 /js/cores/ 里面


2. 在 /js/cores/ 里面添加 seajs-config.js 文件：

seajs.config({
	base: '/js/',
	alias: {
		'jquery': 'cores/jquery-1.11.1.min.js'
	},
	preload:[
		'jquery' // 预加载 jQuery
	]
});

3. 在需要的页面引入 seajs 和 seajs-config 文件，如在 index.html 的 header：

<script type="text/javascript" src="/js/cores/sea.js"></script>
<script type="text/javascript" src="/js/cores/seajs-config.js"></script>

最基本的类库已经准备完毕。
