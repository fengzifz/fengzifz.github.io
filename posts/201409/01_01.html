第一章：文件结构

文件结构如下，主要在 js/ 目录下区分开第三方类库和自己写的模块

injs
  |- js/
  |  |- cores/ 存放第三方类库 JavaScript
  |  |- modules/ 存放自己的 JavaScript 模块
  |- css/ 样式文件
  |- view/ 视图文件
  |- index.html

从 github 上克隆代码（部分目录不显示是因为里面没有文件）：

git clone https://github.com/fengzifz/injs.git
git checkout chapter_01
